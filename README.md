# Fallout Theme for CTFd

A retro-futuristic Fallout-themed CTFd theme inspired by the Pip-Boy interface and Vault-Tec aesthetics. Features a dark terminal-style design with iconic Pip-Boy green (#00FF41), Vault-Tec blue accents, and monospace fonts for that authentic post-apocalyptic terminal feel.

**Theme Features:**
- Pip-Boy green terminal aesthetic with dark backgrounds
- Vault-Tec blue and amber accents
- Monospace fonts (Share Tech Mono) for authentic terminal look
- Glowing green text effects and borders
- Retro-futuristic 1950s Fallout-inspired design
- CRT-style scanline effects (from original Pixo theme)
- Nuclear green highlights and hover effects

Compatible with CTFd **Version 3.3.0**

### Color Palette
- **Pip-Boy Green**: `#00FF41` - Primary text and borders
- **Nuclear Green**: `#39FF14` - Hover states and highlights
- **Vault-Tec Blue**: `#0066CC` - Buttons and info elements
- **Pip-Boy Amber**: `#FFB000` - Warning states
- **Dark Background**: `#0a0a0a` - Main background
- **Terminal Background**: `#1a1a1a` - Card and panel backgrounds

### Installation Steps:
Open your Docker container's terminal then insert the following Command:
```
git clone <your-repo-url> /opt/CTFd/CTFd/themes/fallout
```
Then Login as Admin and go to: ```Admin Panel > Config > Themes``` and switch the Theme to `fallout` and Click on Update.

That's it! Welcome to the wasteland, Vault Dweller!
